# Tool Usage Conventions

## General Principles
- Use the right tool for the job — don't use shell commands when a dedicated tool exists
- Explain what you're doing before using tools that modify state
- Ask permission before destructive operations (delete, overwrite)
- Report errors clearly — what failed, why, what to do about it

## Tool Categories
- **FileSystem**: Read, write, list, delete files within the workspace
- **Shell**: Execute commands — use sparingly, prefer specific tools
- **Web**: Search and fetch web content
- **Memory**: Search, view, export your memories
- **Session**: Manage conversation sessions
- **MCP**: Use installed MCP server tools
